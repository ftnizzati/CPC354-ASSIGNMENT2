Keyboard Shortcuts

  • Q / A = Base (10° / 1° precision)
  • W / S = Main arm
  • L / D = Forearm
  • R / F = Grip
  • G = Record mode
  • X or Space = Save position
  • P = Play mode (loop)
  • Caps Lock = Precision mode
  • + / - = Change movement speed

Arm Control

  • A / D = Base rotation
  • W / S = Lower arm
  • R / F = Upper arm
  • O / P = Gripper open/close

Automation

  • 1 = Start Automation
  • 2 = Stop Automation
  • 3 = Reset Arm
  • 4 = Open Gripper
  • 5 = Close Gripper
  • 6 = Toggle Object Pick

Recording

  • G = Toggle Record mode
  • X / Space = Save position
  • V = Play mode (loop)
  • B = Restore last saved position

Settings

  • Caps Lock = Precision mode
  • + / - = Change speed
